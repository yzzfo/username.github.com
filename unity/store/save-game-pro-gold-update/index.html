      Save Game Pro 是一个完整、强大且功能丰富的存档解决方案，适用于 Unity（游戏引擎），让您可以在任何地方保存所有内容，例如组件、数据类型、游戏对象（包括自定义数据类型），包括 Web 和云、本地存储、PlayerPrefs、数据库。

      Save Game Pro 具有优雅且完整的 API，用于控制保存的数据，例如，通过使用 API，您可以检查数据是否存在，或者您可以检索保存的文件列表并将其显示给用户。

      与 Unity 2017.x 及更高版本兼容，您也可以使其与旧版本的 Unity 一起使用，只需在导入包之前忽略警告即可。

      在以下 Unity 版本上进行了测试：

      - 2018.2, 2018.1, 2017.4, 2017.3, 2017.2, 2017.1, 5.6

      但它也与旧版本和新版本良好兼容，因此如果您在导入包时遇到任何问题，请随时与我们联系。

      ☀️ 支持200+种类型：Save Game Pro支持内置的200+种类型，您可以手动添加自己的额外类型或使用类型创建器。


      *️ 几乎所有组件，如 BoxCollider、Rigidbody、MeshRenderer、Camera、Transform。

      *️ 游戏对象，整个游戏对象将被保存，这意味着所有组件包括所有子对象。

      免责声明，游戏对象的保存和加载功能正在进行大量开发，以支持保存复杂对象和层次结构，因此在此之前，我们不建议您使用游戏对象的保存和加载功能来��存复杂对象，仅限于简单对象。此外，在所有情况下使用数据结构是更好的实践。

      *️ 带层次结构的变换，这意味着变换是通过其根父对象保存的，并将以这种方式加载。

      *️ 基本类型，如整数、字符串、布尔值等。

      *️ 集合，几乎支持所有集合，如字典、列表、链表、多维数组（没有限制）、栈、队列、哈希集等。

      *️ 数据类型，几乎所有数据类型，如 Vector3、Vector2、Vector4、Mesh 等。

      ☀️ 自定义类型（类型创建器）: Save Game Pro 支持自定义类型，这意味着您可以添加 Save Game Pro 不支持的类型的序列化。因此，您可以轻松且自动地使用类型创建器创建这些自定义类型。类型创建器提供了一个简单、快速且实用的界面，用于创建和浏览类型。此外，类型创建器还包括搜索输入，帮助您通过搜索所需类型来浏览所有类型。
