      FEATURES

      1) No need for export & import processes to create your world anymore.

      • Edit and manipulate 3D meshes directly within Unity, in the final environment context.

      • See shader/VFX results instantly whenever you modify a base mesh.

      • Modify and tweak meshes made with external DCC software (such as Blender, 3ds Max, Maya, etc.) right in Unity.

      2) Sketch-Based 3D Modeling Technique

      • Various drawing tools and the PushPull tool allow for intuitive 3D shape creation.

      • Quickly prototype your level using this feature.

      3) Comprehensive 3D Modeling Tools and UV Editor

      • Quickly create 3D assets and levels with nearly 150 versatile tools.

      • UModeler features adjustable 3D primitive shapes for added flexibility.

      • Unwrap any mesh and edit UVs directly in Unity with the built-in UV Editor.

      • New Feature: Hotspot texturing is now available!

      4) High Compatibility with other Unity Assets

      • ProBuilder: Smooth conversion of ProBuilder meshes.

      • Polybrush: Seamlessly edit UModeler meshes with Polybrush.

      • Synty Studio's 3D Assets: Convert 3D meshes into UModeler and edit them seamlessly.

      • Surforge: Fully compatible with PBR texturing.

      • And more

      5) Extensive Learning Materials and Active Online Communities

      • Narrated tutorial videos for both beginners and intermediate users.

      • Comprehensive 'Timeline' and 'Full-making' videos demonstrating 3D asset modeling with UModeler.

      • Engage with a vibrant community on Discord and the Unity Forum. Ask questions, share insights, or showcase your art.
