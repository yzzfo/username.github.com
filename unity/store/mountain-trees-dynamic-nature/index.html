      • Mountain Trees - Dynamic Nature

      This pack contains 8 PBR mountain fir trees in 63 prefabs. Trees could be placed in a forest (special forest tree versions) or as standalone. Pack of shaders contains dynamic snow cover.

      该资源包内含有8种PBR山脉冷杉，有63个预制件。（39个标准预制件+24个Vegetation Studio预制件）。树木可放置在森林里（森林树木特别版本）或单独放置。所有树木都与 Unity 地形和 Vegetation Studio 系统完美适配（需准备额外的预设）。均经过精心优化，根据扫描数据创建。树木的过度绘制、绘制调用和渲染耗能都很低。所有树木均包含非常高级的风着色。树木可被积雪动态叠加，该功能可让你在静态场景中调整树上的积雪数量。

      我们准备了6种树木预制件：
      - 普通：4到8棵树有2种树皮材质，树皮上混合极佳
      - 低耗：所有树木都只有1种树皮材质（更少绘制调用），所有树木都使用相同的树叶和树皮。这意味着8种树木将同时进行批处理（整个森林的绘制调用都很低）。
      - 普通动态雪
      - 低耗动态雪
      - 普通静态雪
      - 低耗静态雪


      所有低耗版树木都是同时进行批处理的，所以即使场景中有 8 种不同的树木，绘制调用数也会大幅降低。

      我们的资源包中包含 4 个方面：

      1）不同阶段的树木，如：
      - 幼苗
      - 小型
      - 中型
      - 2个大型独立树木
      - 2 个森林树木（地面上三角形和阴影更少）
      - 枯树

      2）树木的纹理组
      - 反射率/Alpha、法线、金属/AO/平滑度进行了压缩，为了更低耗的 GPU 渲染和更少的内存占用；
      - 雪设置（5个纹理）
      - 14个树皮套件（45个纹理）
      - 叶子（4个纹理）
      - 16个交叉纹理套件（32个纹理）

      3）着色器：
      - 所有着色器均支持Vegetation Studio间接实例化，立即；
      - 所有着色器均支持 Vegetation Studio Pro 间接实例化，立即可用；
      - 2个交叉着色器（无雪、雪），
      - 4 个树皮着色器（金属、金属雪、高光、高光雪）
      - 4个树叶着色器（金属、金属雪、高光、高光雪）
      - 雪是动态的，所以树木可在运行时被覆盖。
      - Vegetation Studio动态雪实时更新需要额外脚本以获得Vegetation Studio副本材质。
      - Vegetation Studio Pro动态雪实时更新即时可用。
      - 你可用2个值（雪量、雪阈值）调整雪量和雪分布。

      4）渲染风着色功能高级且低耗；
      - 该资源包1.5.v及以上版本都支持全新的风系统，效果极其出色。
      - 风会在所有资源间共享
      - 风在HD SRP中以1:1运行

      注意：我们建议在森林中使用森林树木，以获取更好的光照和性能；单独树木请用在森林边界上，偶尔可用在森林内部。此设置会呈现最佳视觉效果和性能。
