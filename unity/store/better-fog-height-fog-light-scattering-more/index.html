      • Better Fog: Height Fog, Light Scattering & More

      为您的游戏增添深度和感觉。 实现更真实的效果或打造真正独特的外观。 改进或完全改变游戏的外观，无需担心透明、水或天空盒。

      Better Fog 是一种后处理效果，提供多种功能，可打造从逼真到风格化的多种外观和风格。 这种效果不是体积解决方案，可确保最佳性能。 它还包括其他功能，例如平面或球体分析世界雾、自定义雾粒子着色器等。 Better Fog 使用 Shader Graph (URP) 和 Amplify Shader Editor（内置）制作，支持透明对象，无需修改其着色器。

      特征：

      屏幕空间光散射 (SSMS)：使用它可以为雾创建更真实的感觉。

      渐变雾色：创建您自己的自定义外观或使用资源中包含的众多渐变之一。

      天空盒雾集成：确保与任何天空盒保持一致的视觉效果。

      基于距离的雾：默认雾，可以选择使用径向距离。

      高度雾：大气高度雾。

      体积混合：使用后处理体积在各种预设之间进行混合。

      透明物体和粒子支持：雾对于粒子和透明物体效果很好。

      世界雾：使用带有自定义着色器的平面和球体自定义并添加雾。

      带粒子的自定义雾：包括程序雾粒子 FX。

      雾能量损失：模拟由于雾散射而导致的光子能量损失。

      简单集成：轻松将效果添加到您的项目中。

      多功能定制：根据需要调整每个功能并使用关键字以获得更好的性能。

      兼容性：

      支持URP和内置版本2021.3及以上。

      限制：

      仅支持定向光。

      非体积。

      不支持 MSAA。

      需要能够进行深度渲染的设备。

      不支持正交相机、VR、webGL、VFX 图形粒子混合。
