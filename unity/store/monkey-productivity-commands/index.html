      ⭐详细信息⭐

      ✔️ 为设计师、艺术家和程序员提供超过130个命令

      MonKey配备了满足所有Unity用户需求的命令。

      它们涵盖的功能包括：

      ◽对象创建

      ◽对象查找

      ◽选择

      ◽重命名

      ◽场景管理

      ◽关卡设计

      ◽预制件

      ◽可脚本化对象

      ◽用户界面

      ◽物理

      ◽组件

      ◽还有更多：查看命令列表！

      ✔️ 带属性的自定义命令

      使用属性以超快且独立于代码的方式编写具有高级自定义和参数支持的新命令。

      只需将属性添加到静态编辑器方法，它将在MonKey中可用！

      ✔️ 超级快速的模糊搜索与出色的性能

      我们的自定义搜索算法提供最佳的人性化体验！

      多种访问命令的方式，包括热键、快速名称和关键字搜索。

      MonKey针对大型项目或大量命令进行了优化。


      ✔️ 集成命令帮助与人性化命名

      MonKey 的界面为所有命令提供帮助，并以易于理解的方式命名。

      ✔️ 热键管理器与快速命名MonKey 检测命令与菜单项之间的热键冲突，并在搜索中显示它们。

      使用快速命名作为热键的快速替代方案！

      ✔️ 参数化命令与命令验证：

      参数化命令完全支持，比 Unity 菜单项提供更直观和高级的用法。

      MonKey 配备自动完成特性，可以快速搜索 GameObjects、Assets、Enums 或任何变量类型。

      对于参数，MonKey 支持 Unity 类型、C# 值类型、枚举、数组等，并且可以轻松扩展！

      MonKey 还具有命令验证功能，以限制命令在特定场景中的使用。

      ✔️ 集成您的插件和 Unity 的菜单项

      直接在 MonKey 中搜索 Unity 的菜单功能和您的插件菜单项！

      ✔️ 用于关卡设计的交互式场景命令

      强大且新兴的命令用于场景视图中的关卡设计。

      结合少量功能以加速您的创作！

      包括碰撞约束、距离约束、对象实例化、测量、对齐、夹紧、组件、对象替换等命令，以及更多功能！

      ✔️ 完整的撤销支持

      MonKey 的命令具有完整的撤销支持（在 Unity 的限制内）。再也不用担心实验了！

      ✔️ 灵活的界面

      界面的用户体验设计旨在易于扩展，并在直观与功能之间取得平衡。MonKey 的控制台可以停靠或用作弹出窗口，并包含许多设置以自定义外观和使用方式。


      ✔️ 有序选择

      MonKey 存储您选择对象的顺序（Unity 不会）：通过使用我们的系统，使您的命令更具用户友好性和可靠性！

      ✔️ C# 编辑器工具：MonKey 包含高级功能和对编辑器变量的访问，例如鼠标位置、简单的撤销设置、场景射线检测、按键事件、层级搜索等。

      让您的编辑器工具创建真正闪耀！

      ✔️ 主动支持和命令路线图

      我们致力于定期添加新命令和功能，以加速您的工作流程，让您充分体验 Unity 编辑器的潜力！
