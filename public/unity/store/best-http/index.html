      • 请求处理灵活性：
        • 传统回调: 简单明了的响应处理。
        • Unity 协程: 与 Unity 工作流集成的非阻塞操作。
        • 异步-等待模式: 用于异步操作的线性非阻塞代码。
      • 高级网络功能：
        • 🔥 连接池，支持 HTTP/1 和 HTTP/2。
        • 🔥 DNS 缓存，具有预取功能。
        • 支持身份验证挑战、尾部头和更多的高级响应处理。
      • 性能调优: 根据应用程序的需求微调资产，调整参数以实现最佳性能和最小资源使用。
      • 🔥 内存缓冲池: 通过缓冲池优化内存管理，减少分配和垃圾回收，尤其在大文件上传或下载场景中实现更流畅的性能。
      • 服务器兼容性: 无需特殊服务器设置，兼容旧版和最新服务器技术。
      • 协议支持：
        • 支持 HTTP 和 HTTPS 协议以实现安全通信。
        • 🔥对 HTTP/2 特性的高级支持，包括用于 GRPC 等场景的 尾部头。
      • 高效数据管理：
        • 🔥 缓存机制: 减少冗余网络请求。
        • 压缩: 使用 Gzip 和 deflate 算法进行带宽优化。
      • 数据传输：
        • 🔥 流式传输: 支持下载和上传的流式传输，以避免内存瓶颈。
        • 📰 表单: url 编码、多部分、json
        • 编码: 分块和原始
      • 请求自定义: 可自定义的头部、超时和其他参数。
      • 🔥 性能分析器集成：
        • 内存分析器: 监控内部内存使用情况。
        • 网络分析器: 具有字节跟踪、连接分析和 DNS 缓存分析等功能的细粒度分析。
      • 调试和日志记录: 提供全面的日志记录选项以获取洞察和调试信息。
      • 🔐 安全性和身份验证：
        • 支持多种 身份验证 方法: 基本、摘要 和 Bearer 令牌。
        • 🍪 Cookie 管理: 为 Web 应用程序提供轻松的 Cookie
